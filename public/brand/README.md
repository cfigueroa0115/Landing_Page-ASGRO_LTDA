# Brand Assets Directory

Place the following brand image files in this directory:

- **asgro-logo.png** — ASGRO company logo (recommended: 200x60px or larger, transparent background)
- **asgro-services-banner.png** — Services section banner image (recommended: 1200x600px)
- **asgro-og-image.png** — Open Graph / social sharing image (recommended: 1200x630px)

## Notes

- These files are NOT committed to version control (add them manually per deployment)
- If any file is missing, the application renders professional SVG/CSS fallbacks automatically
- Use PNG format with transparent backgrounds where appropriate
- Ensure the logo is legible at small sizes (minimum 40px height)
